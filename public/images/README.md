# TransNext Logistik - Bilder

## Hero-Bild für die Startseite

**Erforderlich:** `hero-transnext.jpg`

Bitte speichern Sie das hochgeladene Bild mit dem TransNext-Mitarbeiter vor der Fahrzeugflotte hier als `hero-transnext.jpg`.

### Wie Sie das Bild hochladen:

1. **Option 1 - Über den Datei-Explorer (empfohlen):**
   - Öffnen Sie den Datei-Explorer in Same (linke Seitenleiste)
   - Navigieren Sie zu `transnext-logistik/public/images/`
   - Klicken Sie mit der rechten Maustaste auf den `images` Ordner
   - Wählen Sie "Upload Files" oder ziehen Sie das Bild per Drag & Drop
   - Benennen Sie die Datei `hero-transnext.jpg`

2. **Option 2 - Über das Terminal:**
   Wenn Sie das Bild bereits auf Ihrem Computer haben, können Sie es via Terminal hochladen oder hier ablegen.

### Bild-Spezifikationen:

- **Dateiname:** `hero-transnext.jpg`
- **Format:** JPG/JPEG (empfohlen) oder PNG
- **Empfohlene Auflösung:** Mindestens 1920x1440 Pixel
- **Seitenverhältnis:** 4:3
- **Dateigröße:** < 2 MB für optimale Ladezeiten
